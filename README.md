# 動画圧縮ツール

MP4・MOV動画ファイルをFFmpegで一括圧縮するWindowsツールです。

## 機能

- ファイルまたはフォルダをドラッグ&ドロップで圧縮
- 複数パスの同時指定
- 圧縮ジョブの並列実行（ドライブ種別に応じて自動調整）
- 4K動画の自動ダウンスケール（10分以上のみ）
- 圧縮失敗時の元ファイル自動復元

## 必要環境

- Windows
- [FFmpeg](https://ffmpeg.org/) （`ffmpeg` および `ffprobe` が PATH に通っていること）

## 使い方

### ドラッグ&ドロップ

`compress_videos.bat` にファイルやフォルダをドラッグ&ドロップします。複数同時にドロップ可能です。

### 直接実行

`compress_videos.bat` をダブルクリックし、パスを入力します。

```
圧縮対象のファイル/フォルダパスを入力してください: C:\Videos
```

複数パスはセミコロン(`;`)またはカンマ(`,`)で区切ります。

```
圧縮対象のファイル/フォルダパスを入力してください: C:\Videos;D:\Footage
```

### コマンドラインから実行

```bat
compress_videos.bat "C:\Videos"
compress_videos.bat "C:\Videos" "D:\Footage\project1"
```

## 動作仕様

| 項目 | 内容 |
|------|------|
| スキップ条件 | 500 MB 以下のファイルは処理しない |
| 対象フォーマット | `.mp4` / `.mov`（サブフォルダも再帰的に処理） |
| エンコード設定 | libx264 / preset medium / CRF 23 / AAC 128k |
| 4Kダウンスケール | 3840×2160以上 かつ 10分以上の動画を 1920×1080 に縮小 |
| 並列数（HDD） | 1〜2ジョブ |
| 並列数（SSD） | CPUコア数 − 1 ジョブ |
| ファイル置換 | 成功時: 元ファイルを削除し `.tmp.mp4` をリネーム |
| 失敗時復元 | 失敗時: `.tmp.mp4` を削除し元ファイルを復元 |
| キャンセル | Ctrl+C で全ジョブ停止・ffmpegプロセス強制終了 |
